## üîç Troubleshooting

### Issue: Build fails with "maturin" or "metadata-generation-failed" error

**Fix**:

1. Ensure `PYTHON_VERSION=3.11.0` is set in environment variables
2. Check that `runtime.txt` file exists with `python-3.11.0`
3. Use build command: `pip install --upgrade pip && pip install --no-cache-dir -r requirements.txt`
4. If still failing, try clearing build cache in Render settings

### Issue: Build fails

**Fix**: Check Render logs ‚Üí Common causes:

- Missing dependencies in requirements.txt
- Syntax errors in Python code
- Import errors

### Issue: App crashes on startup

**Fix**:

1. Check environment variables are set correctly
2. Verify SECRET_KEY is generated (not default value)
3. Check GROK_API_KEY is valid

### Issue: CORS errors in browser

**Fix**:

1. Update ALLOWED_ORIGINS with your actual Render URL
2. Format: `https://your-app-name.onrender.com` (no trailing slash)
3. Save ‚Üí Will trigger auto-redeploy

### Issue: WebSocket not connecting

**Fix**:

1. Ensure using `wss://` (secure WebSocket)
2. Check browser console for errors
3. Verify CORS settings

### Issue: "Service Unavailable" after 15 min

**This is normal for free tier:**

- Service spins down after 15 min of inactivity
- Takes ~30 seconds to wake up (cold start)
- Upgrade to paid plan for 24/7 uptime

## üí° Pro Tips

### Get Groq API Key (Free):

1. Visit https://console.groq.com/
2. Sign up (free)
3. Go to API Keys section
4. Create new key
5. Copy and use in GROK_API_KEY

### Upgrade to PostgreSQL (Recommended):

1. Render Dashboard ‚Üí **"New +"** ‚Üí **"PostgreSQL"**
2. Copy connection string
3. Update DATABASE_URL env variable
4. Add to requirements.txt: `asyncpg==0.29.0`

### Custom Domain:

1. Render Dashboard ‚Üí Your Service ‚Üí Settings
2. Click **"Custom Domain"**
3. Add your domain (e.g., `taskassistant.com`)
4. Update DNS records as shown

### Keep App Active (Free Tier):

- Use UptimeRobot or similar
- Ping health endpoint every 14 minutes
- Note: Uses more free hours
